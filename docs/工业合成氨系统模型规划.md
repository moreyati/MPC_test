# 工业合成氨系统模型规划（参数设定）

## 1. 模型定位与接口

- **位置**：接在 PID 输出之后；PID 输出流量 → 本模型输入；本模型输出温度。
- **输入**：流量 \(Q\)，单位 Nm³/h，范围 **0 ~ 60 000 Nm³/h**。
- **输出**：温度 \(T\)，单位 ℃，范围 **0 ~ 600 ℃**。
- **设计工作点**：**50 000 Nm³/h 对应 500 ℃**（稳态）。

## 2. 稳态映射（流量 → 温度）

目标：在稳态下，流量与温度一一对应，且具有非线性（物料越多，同等流量增量带来的温升更大，即“物料越多升温越快”的稳态体现）。

### 2.1 参数建议

| 参数名 | 含义 | 建议范围/取值 | 说明 |
|--------|------|----------------|------|
| \(Q_{ref}\) | 参考流量 (Nm³/h) | 50 000 | 设计点流量 |
| \(T_{ref}\) | 参考温度 (℃) | 500 | 设计点温度 |
| \(Q_{min}, Q_{max}\) | 流量范围 | 0, 60 000 | 与 PID 一致 |
| \(T_{min}, T_{max}\) | 温度范围 | 0, 600 | 输出限幅 |

### 2.2 稳态关系形式（可选）

- **方案 A（幂函数）**：\(T_{ss} = T_{min} + (T_{max}-T_{min}) \cdot (Q/Q_{max})^n\)，\(n>1\) 时大流量段更陡（非线性）。
- **方案 B（设计点线性 + 非线性修正）**：\(T_{ss} = T_{ref} \cdot (Q/Q_{ref})^n\)，再限制在 \([T_{min}, T_{max}]\)，\(n\) 略大于 1 表示“物料多时更敏感”。
- **方案 C（查表/分段线性）**：若干 \((Q_i, T_i)\) 点，设计点 (50000, 500)，低流量段斜率小、高流量段斜率大，插值得到 \(T_{ss}(Q)\)。

设计点约束：\(Q=50\,000 \Rightarrow T_{ss}=500\)，需在公式或表中显式满足。

## 3. 动态特性（非线性）

要求：**物料越多升温越快** —— 即高流量时，温度变化更快（时间常数更小或等效增益更大）。

### 3.1 参数建议

| 参数名 | 含义 | 建议范围 | 说明 |
|--------|------|----------|------|
| \(\tau_{base}\) | 基准时间常数 (s) | 10 ~ 30 | 设计点附近一阶惯性 |
| \(\alpha\) | 流量对时间常数的影响指数 | 0.3 ~ 0.8 | \(\tau(Q) \propto 1/Q^\alpha\) 或类似，使大流量时 \(\tau\) 变小 |
| 或 \(k_{gain}(Q)\) | 增益随流量变化 | 随 \(Q\) 增而增 | 替代或配合 \(\tau(Q)\)，实现“大流量时响应更快” |

### 3.2 动态形式（可选）

- **一阶惯性**：\(\tau(Q)\, \frac{dT}{dt} + T = T_{ss}(Q)\)，其中 \(\tau(Q)\) 随 \(Q\) 增大而减小（物料多、升温快）。
- **非线性增益**：\(dT/dt = [T_{ss}(Q)-T] / \tau(Q)\)，\(\tau(Q) = \tau_{ref} \cdot (Q_{ref}/Q)^\alpha\)（\(\alpha>0\)），保证 \(Q\) 大时 \(\tau\) 小。

## 4. 参数汇总表（实现前可填具体数值）

| 符号 | 名称 | 单位 | 建议值/范围 | 备注 |
|------|------|------|-------------|------|
| \(Q_{min}\) | 最小流量 | Nm³/h | 0 | 输入下界 |
| \(Q_{max}\) | 最大流量 | Nm³/h | 60 000 | 输入上界 |
| \(T_{min}\) | 最低温度 | ℃ | 0 | 输出下界 |
| \(T_{max}\) | 最高温度 | ℃ | 600 | 输出上界 |
| \(Q_{ref}\) | 参考流量（设计点） | Nm³/h | 50 000 | 对应 500℃ |
| \(T_{ref}\) | 参考温度（设计点） | ℃ | 500 | 设计点输出 |
| \(n\) | 稳态非线性指数 | - | >1（如 1.2~1.5） | 大流量段更陡 |
| \(\tau_{ref}\) 或 \(\tau_{base}\) | 参考/基准时间常数 | s | 15~25 | 设计点附近动态 |
| \(\alpha\) | 时间常数随流量变化指数 | - | 0.3~0.6 | 大流量时 \(\tau\) 减小 |

## 5. 实现时注意事项

1. **单位统一**：内部仿真步长若为 s，流量 Nm³/h 若参与动态，需按 3600 换算或保持一致性。
2. **限幅**：输入 \(Q\) 限制在 \([0, 60\,000]\)，输出 \(T\) 限制在 \([0, 600]\)。
3. **初值**：建议与设计点一致，如 \(Q(0)=50\,000 \Rightarrow T(0)=500\)，便于与 PID 初始设定值 5 万 Nm³/h 对接。
4. **先稳态后动态**：先实现并校验 \(T_{ss}(Q)\) 满足 50 000→500 与 0~600 范围，再叠加 \(\tau(Q)\) 动态。

## 6. 后续代码接入点（规划用，暂不写代码）

- 在 PID 仿真循环中，当前 **PID 输出** 即 **流量 \(Q\)**（0~60 000 Nm³/h）。
- 每一仿真步：用当前 \(Q\) 计算 \(T_{ss}(Q)\)，再用 \(\tau(Q)\) 一阶更新 \(T\)，得到 **模型输出温度**。
- 前端可增加“温度”曲线与显示；若仍只做流量 PID，则本模型仅作后端计算与记录，为后续“温度-PID”或“流量-温度串级”预留。
